Quick prototyping

Just draw your schematic with standard tools (like Illustrator), reuse PDF files, upload it to WILCO and it gets animated with your data!

Proprietary data connectivity

WILCO layout helper allows you to describe your data and map it to parameters. For complex data, you can create your own adapter using the WILCO API. WILCO comes with some plugins to read CSVs, ACARS data and more...

on the fly analysis and alerts

We call them IFTs (If This Then That). Create simple rules that runs each time a data is received and do something on event: send an e-mail, a SMS, command your system, make a request and more... The IFTs can be activated or desactivated in one click!

Latest WEB technologies

WILCO uses cutting edge technologies to bring to the live system status to the technicians or engineers in the browser, a tablet or a phone without any clientside installation

Stunning dashboards

The dashboards are interactives. Click on a gauge, you have access to all the relevant graphs. Click on a dot, WILCO brings you to the context of the dot. get access to the infinite timeline of events and filter.

Secured access

WILCO is based on secured HTTS access. You can manage your users, the access rights using the User admin page

Easy integration

Wilco is open. all the data is accessible using a restful JSON API. you can integrate it into your information system to push or pull information.

Some realizations in aeronautics

We have connected the aircraft! We made fleet monitoring, predictive maintenance and turnaround KPIs live and easy without installing anything on-board.
